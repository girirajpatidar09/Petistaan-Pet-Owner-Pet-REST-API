## Description

A Spring Boot RESTful API for managing Pet Owners and Pets. Built using Java 17, Spring Boot, Spring Data JPA, Hibernate, and MySQL with DTO pattern, validation, centralized exception handling, and Swagger documentation.

## Features

CRUD operations for Pet Owners and Pets

One-to-one bidirectional JPA mapping

DTO pattern with custom mappers

Centralized exception handling

Bean Validation with custom messages

Swagger/OpenAPI documentation

MySQL database integration

Lombok for reducing boilerplate
